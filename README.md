# Тестовое задание для ООО «Экосистема Альфа»

Проект представляет собой SPA со списком карточек (продукты). Данные берутся с портала роскачества (https://rskrf.ru) для категории кетчупов.

## Использованные технологии

Typescript, React, Redux, Vite

## Функционал

- Загрузка продуктов через API, сохранение их в сторе
- Добавление/удаление продуктов в избранное
- Удаление продуктов (только из стора, т.к. API не позволяет удалять продукты)
- Добавление новых продуктов (только в стор, т.к. API не позволяет добавлять продукты)
